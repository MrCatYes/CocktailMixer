# Cocktail Mixer website

## Guide d'installation

1. Cloner ou télécharger le code
2. Créer un fichier .env contenant les informations de connection à la base de donnée:
   ```
    DB_HOST=
    DB_PORT=
    DB_USERNAME=
    DB_PASSWORD=
    DB_DATABASE=
   ```
3. Installer les dépendances dans le backend et démarrer le serveur :
   ```
    cd .\backend\
    npm i
    npm start
   ```
4. Installer les dépendances dans le frontend et démarrer le serveur :
   ```
    cd ..\frontend\
    npm i
    npm run dev
   ```
5. Accéder au site http://localhost:3000
